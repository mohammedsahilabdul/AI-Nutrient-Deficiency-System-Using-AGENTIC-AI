```bash

```

### 2. Add Your API Keys (Get free GROQ key)
```env

GROQ_API_KEY=gsk_YOUR_KEY_HERE
SENDER_EMAIL=your@gmail.com
SENDER_PASSWORD=app-password
```

### 3. Run the App
```bash
run_streamlit.bat
```

### 4. Open Browser
Go to: `http://localhost:8501`

## 🆘 QUICK TROUBLESHOOTING

| Issue | Solution |
|-------|----------|
| Port already in use | Kill process on 8501 or wait |
| .env not found | Copy `.env.template` to `.env` |
| API key error | Check key format and .env syntax |
| Email not sent | Verify Gmail App Password |
| Slow analysis | Normal first run, try Groq |
| Images not upload | Use JPG/PNG, <10MB size |
