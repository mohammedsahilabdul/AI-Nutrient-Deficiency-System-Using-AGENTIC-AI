## 📞 SUPPORT & TROUBLESHOOTING

### Quick Fixes
```bash
# Server won't start
python main_new.py

# Check health
curl http://localhost:8000/health

# Reset system
curl -X POST "http://localhost:8000/api/reset" \
  -H "X-API-Key: YOUR_ADMIN_KEY"

# Clear cache
curl -X POST "http://localhost:8000/api/cache/clear" \
  -H "X-API-Key: YOUR_ADMIN_KEY"

# View logs
tail -f server.log
```

### Common Issues
```
Server won't start       → Check Python 3.9+, dependencies
API returns 500         → Check logs, run verify_system.py
Database issues         → Run init_db(), check permissions
Auth failing            → Regenerate API keys
Slow performance        → Clear cache, check queries
PDF not generation      → Install reportlab
Email not sending       → Configure Gmail credentials
```
